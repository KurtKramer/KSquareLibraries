Detailed Description

Will be one TrainingConfiguration2::Factory derived class for each derived class of "TrainingConfiguration2"; they will be responsible for creating an instance of the correct "TrainingConfiguration2" derived instance. The static member "TrainingConfiguration2::registeredFactories" will keep track of all instances of the Factory derived class.
