KKStr KKStrParser::GetNextToken ( const char *  delStr = "\n\t\r ")

Extract next Token from string, each delimiter will separate tokens.

Removes next Token from string. The token will be terminated by end of string or the first occurrence of a delimiter character. If no more tokens left will return a Empty KKStr.

Parameters
[in]delStrList of delimiting characters.
Returns
Extracted Token.
