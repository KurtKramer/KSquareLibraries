Detailed Description

Namespce used to wrap implementation of libSVM version 2.89.

There is more than obe version of libSVM implemented in the library. To prevent name coflicts between them each one was wrapped in their own namespace.
libSVM is a Support Vector Machine implementaton done by "Chih-Chung Chang" and "Chih-Jen Lin". It was downloaded from http://www.csie.ntu.edu.tw/~cjlin/libsvm/. The source code was modified by Kurt Kramer. The primary changes to this implementation involves the replacement of the sparce datastructure in the original implementation with fixed length array implemented through the "FeatureVector" class and the ability to specify a sub-set of features to be utilized via the "FeatureNumList" class. This allows us to load in a single set of training data with all its features that can then beuswed for multuple Support Vector Machine instances where each instance utilizes a different set of features. The use of this version of libSVM(SVM289_MFS) is via the "ModelSvmBase" class.
