8 /**
9  *@class KKMachineLearning::FeatureNumList
10  *@brief Keeps track of selected features.
11  *@details Used by SVMModel and the newer 'Model', 'ModelParam' based classes.
12  * Each instance of this class will have an associated 'FileDesc' instance. It
13  * is meant to keep track of selected features from that instance('FileDesc').
14  * <br />
15  * This class is meant to work with both the FeaureVector, Model, and ModelParm
16  * based classes. This will allow us to select specific features
17  * from the FeatureVector instances that are to be used.
18  * <br />
19  * At no time are feature numbers that are out of range of the associated
20  * FileDesc instance to be selected.
21  * <br />
22  * Two of the constructors allow you to specify a list of features in a string.
23  * The list can consist of single features and/or ranges of features. Ranges of
24  * features are specified by using the dash('-') charter between two numbers.
25  * The comma(',') character will be used as a separator. "All" specifies all are
26  * to be selected except those that are flagged as 'IgnoreAttribute' in the
27  * Associated FileDesc instance. The list should be in ascending order.
28  *
29  *@code
30  * Example Strings:
31  * "1,2,3,10,20" Selects [1,2,3,10, 20].
32  * "1,4-7,9-12,13" Selects [1,4,,5,6,7,9,10,11,12,13]
33  * "All" Selects all features that '_fileDesc' includes accept
34  * those that are flagged as 'IgnoreAttribute' in the
35  * associated FileDesc instance.
36  *@endcode
37  */
