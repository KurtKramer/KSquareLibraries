Constructor that will extract a list of feature vectors for all the image files in the specified directory.

Will scan the directory _dirName for any image files. For each image found a new instance of PostLarvaeFV will be created whos features will be derived from the image. These PostLarvaeFV' objects will be assigned the class specified by '_mlClass'. A new data file containg the extracted features will be saved in fileName.

Parameters
_log[in]Log file to write messages to.
_mlClass[in]Class to assign to new 'PostLarvaeFV' objects.
_dirName[in]Directory to scan for examples.
_fileNameName of file to contain the extracted feature data. Will be og the Raw format.

PostLarvaeFVList::PostLarvaeFVList ( const FeatureVectorList &  featureVectorList)

This constructor is meant to create a list of 'PostLarvaeFV' objects from the FeatureVector objects contained in featureVectorList.

If 'featureVectorList' owns its contents (that is 'featureVectorList.Owner () == true' then
| We will create new Instances of 'PostLarvaeFV' objects that we will own.
| The underlying class of the 'FeatureVector' objects will be converted to a
| 'PostLarvaeFV' class.
else
| all the 'FeatureVector' objects in 'featureVectorList' must have an underlying class of
| 'PostLarvaeFV'. If one or more do not then the program will halt with a message to
| the log.
